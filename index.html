Présentation

Après plusieurs années dans le domaine de l'audiovisuel en qualité d'encadrant technique dans la maintenance, l'installation et le conseil pour le Home Theater, je me passionne aujourd'hui pour la Programmation et plus particulièrement le Web.
J'ai appris les bases du langage de programmation à la 3W ACADEMY, et de manière autodidacte en prenant appui auprès d'Udémy. Puis j'ai approfondi mes connaissances avec OpenClassRooms en validant un titre de niv 5 (bac +2) de Développeur web.

Langages

HTML 5

Le langage HTML (de l’anglais Hypertext Markup Language) est, depuis les toutes premières heures de l’Internet, le programme de base en matière de structuration.

CSS 3

Les feuilles de style en cascade, appelées CSS (de l'anglais Cascading Style Sheets), forment un langage informatique qui décrit la présentation des documents HTML.

Bootstrap

Bootstrap est une collection d'outils utiles à la création du design de sites et d'applications web. C'est un ensemble de codes HTML et CSS, ainsi que des extensions JavaScript en option.

Javascript

Gérer la logique d'un programme en JavaScript, récupérer et modifier des éléments du DOM, Utiliser des méthodes asynchrones.

Node.js

Node est le runtime qui permet d'écrire toutes les tâches côté serveur, en JavaScript, telles que la logique métier, la persistance des données et la sécurité.

Vue.js

Construire une application Vue.js, créer des composants réutilisables dans Vue.js, gérer les données d'une application.

MongoDB

Base de données NoSQL, pour rendre une API entièrement dynamique et les données pleinement persistantes.

MySQL

Configurer MySQL, créer et gérer des bases de données.

WordPress

WordPress est un logiciel conçu pour toutes et tous, mettant l’accent sur l’accessibilité, la performance, la sécurité et la facilité d’utilisation.
